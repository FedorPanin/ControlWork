Описание решения:

1. Пользователь вводит элементы массива через запятую с клавиатуры.
2. Введенная строка разбивается на элементы массива с помощью функции `Split(',')`, которая использует запятую в качестве разделителя.
3. Создается новый пустой массив `newArray` с той же длиной, что и исходный массив.
4. Инициализируется переменная `newIndex` со значением 0.
5. Выполняется цикл, который проходит по каждому элементу исходного массива:
   - Если длина текущего элемента меньше или равна 3 символам:
     - Текущий элемент добавляется в массив `newArray` на позицию `newIndex`.
     - Значение переменной `newIndex` увеличивается на 1.
6. Создается новый массив `finalArray` размером `newIndex`.
7. Используя функцию `Array.Copy`, элементы из `newArray` копируются в `finalArray`.
8. Выводится сообщение "Новый массив:".
9. Выполняется цикл, который проходит по каждому элементу массива `finalArray` и выводит его на экран.
10. Программа завершается.

В результате выполнения программы будет сформирован новый массив, содержащий только те элементы, длина которых меньше или равна 3 символам, и этот массив будет выведен на экран.
